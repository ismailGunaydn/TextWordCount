## ✍️ About TextWordCount

**TextWordCount** is a free, privacy-first online word counter and comprehensive text analysis platform with built-in AI writing tools — trusted by 10,000+ writers, students, and professionals daily.

Count words, characters, sentences, paragraphs, syllables, and unique words in real time as you type. Calculate reading time, speaking time, handwriting time, and reading level. Analyze keyword density for SEO. Fix grammar, paraphrase, optimize readability, and stylize content tone — all in one tab, all in your browser.

> **Our philosophy:** A word counter should do more than count words. It should give writers the context to make their content better — instantly, privately, and for free.

## 🤖 AI Writing Features

Four AI tools built directly into the editor — no separate tabs, no copy-pasting, no additional accounts:

### Inline AI Tools
| Tool | What It Does |
|------|-------------|
| **Paraphrase** | Rewrite selected text with different wording while preserving meaning |
| **Fix Grammar** | Correct spelling, punctuation, subject-verb agreement, tense consistency |
| **Optimize** | Simplify complex sentences, reduce passive voice, improve clarity |

### Style Conversion
| Style | Best For |
|-------|---------|
| **Formal** | Business emails, client proposals, executive summaries |
| **Casual** | Blog posts, social media, newsletter intros |
| **Academic** | Research papers, thesis writing, literature reviews |
| **Creative** | Storytelling, marketing copy, brand voice content |

### How AI Tools Work
```
1. Select text in the editor
2. Click an AI button (Paraphrase / Grammar / Optimize / Stylize)
3. AI processes only the selected text — not your entire document
4. Result replaces the selection instantly
5. All 20+ metrics auto-update immediately
```

**Privacy note:** AI tools require a network call for processing. Only the selected text is sent — never stored before, during, or after. The call is stateless.

### Key Architecture Decisions
- **Single-pass text analysis** — one character-by-character loop extracts 20+ metrics simultaneously
- **Client-side only processing** — zero POST requests carrying text content
- **Service Worker** — offline capability after initial page load
- **Tiered debouncing** — instant metrics on every keystroke, heavier analysis on 150ms pause
- **No external dependencies** for text analysis — pure TypeScript functions
- **Sub-1ms processing** for texts up to 50,000 words

## 🔒 Privacy & Trust

### ✅ What We Do
- Process every character in your browser — nothing uploaded
- Store settings only in browser localStorage
- Provide AI tools with stateless, non-persistent calls
- Send only selected text (not full document) for AI processing
- Disclose affiliate links and advertising clearly
- Work offline via Service Worker

### ❌ What We Don't Do
- We do **not** store, transmit, or read your text on any server
- We do **not** require accounts, email, or registration
- We do **not** track your writing activity or content
- We do **not** sell, rent, or share any user data
- We do **not** gate features behind paywalls
- We do **not** use dark patterns or fake urgency

### How to Verify
Open DevTools → Network tab → type anything. **Zero POST requests** carrying your text. The only network calls are static assets on page load and optional AI tool requests when you explicitly click an AI button.
